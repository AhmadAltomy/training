## Training 

## Create a project

### 1. Create a project

- Create Project with `vite`
```bash
npm init vite@latest
```
- Select `react-ts`
- Select `yarn` or `npm`


### 2. Install dependencies
`npm install` or `yarn install`


### 3. Run the project
`npm run dev` or `yarn dev`


### 4. Build the project
`npm run build` or `yarn build`
